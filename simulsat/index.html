Sign Convention

Northern Hemisphere:
Negative roll = East end of Simulsat is elevated
Positive roll = West end of Simulsat is elevated

Southern Hemisphere:
Opposite convention (per ATCi note)

Formulas

Roll Angle: gamma = arctan( sin(delta_lambda) / tan(latitude) ) delta_lambda = Arc_Center_Lon - Site_Lon (positive = arc center is East of site) Elevation to Centerline Satellite: cos(g) = cos(delta_lambda) * cos(lat) El = arctan( (cos(g) - 0.1513) / sqrt(1 - cos(g)^2) ) 0.1513 = Earth_radius / GEO_orbit_radius Azimuth to Centerline Satellite: Az = 180 + arctan( tan(delta_lambda) / sin(lat) ) Verified against ATCi data: Chennai 12.65N, 80.14E, Arc 104.5E -> Roll = 61.4 (calc: 61.5) Chennai 13.13N, 80.17E, Arc 89.5E -> Roll = 33.3 (calc: 34.8)
